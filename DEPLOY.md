# Deployment Guide - AI Affiliate Storefront

## Quick Deploy (Vercel)

1. **Install Vercel CLI** (if not already):
   ```bash
   npm install -g vercel
   ```

2. **Login to Vercel**:
   ```bash
   vercel login
   ```

3. **Deploy**:
   ```bash
   cd memory/ai-affiliate-storefront
   vercel --prod
   ```

## Alternative: GitHub Pages

1. Create a new GitHub repository
2. Push the contents of `memory/ai-affiliate-storefront/`
3. Enable GitHub Pages in repository settings

## Files Ready for Deployment
